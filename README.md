## install and run
```
$ git clone https://github.com/gruppe7/admin2.0
$ npm install
$ npm start
```

## Testing with Jasmine/Karma

```
# Install Karma:
$ npm install karma --save-dev

# Install plugins that your project needs:
$ npm install karma-jasmine karma-chrome-launcher --save-dev

# Run tests
$ npm test
```
